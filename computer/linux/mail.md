## install
Install mail command on CentOS/Redhat 7/6/5:
```
yum install mailx
```
Install mail command on Fedora 22+:
```
dnf install mailx
```
Install mail command on Ubuntu/Debian/LinuxMint:
```
sudo apt-get install mailutils
```
